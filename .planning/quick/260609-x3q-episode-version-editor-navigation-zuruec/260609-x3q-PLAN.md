    **Änderung 1 — Action-Bar (Zeile ~814–817):**
    Nach dem bestehenden `<Link href={backHref} className={styles.secondaryButton}>Zurück</Link>`
    folgenden Block einfügen:
    ```
    {fansubGroupHref != null ? (
      <Link href={fansubGroupHref} className={styles.secondaryButton}>
        Zur Fansubgruppe
      </Link>
    ) : null}
    ```
    Das Button-Primitive `Button` aus @/components/ui könnte statt `<Link>` genutzt werden,
    da es `href` unterstützt und `variant="secondary"` die gleiche Optik liefert.
    Da der bestehende Link aber das lokale `.secondaryButton`-CSS verwendet (kein globales
    Button-Primitive), bleibt die Konsistenz gewahrt, wenn derselbe Ansatz verwendet wird:
    `<Link href={fansubGroupHref} className={styles.secondaryButton}>`.

    **Änderung 2 — Header-Subtitle (Zeile ~219–224):**
    Die aktuelle Zeile:
      `{groupName ? \` · ${groupName} ${segmentVersion}\` : ""}`
    ersetzen durch:
      ```jsx
      {groupName ? (
        <>
          {" · "}
          {fansubGroupHref != null ? (
            <Link href={fansubGroupHref} className={styles.subtitleGroupLink}>
              {groupName}
            </Link>
          ) : (
            groupName
          )}
          {" "}
          {segmentVersion}
        </>
      ) : null}
      ```
    Damit bleibt der Text „Episode 001 · [SubGroup-Link] v1" erhalten; nur der Gruppenname
    wird zum Link. `segmentVersion` bleibt als Plaintext daneben.

    ```css
    }
    ```
    Die `.subtitle`-Klasse (Zeile 50–54) definiert bereits `color: #6b6b70`; `color: inherit`
    greift diesen Wert auf, sodass der Link optisch eingebettet wirkt.
