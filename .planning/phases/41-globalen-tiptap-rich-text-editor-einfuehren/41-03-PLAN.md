    **Schritt 3: Handler-Logik auf TipTap umstellen**

    In CreateFansubGroupNote und UpdateFansubGroupNote (und Äquivalenten):
    ERSETZE: `h.markdownSvc.RenderMarkdown(req.BodyMarkdown)`
    DURCH:
    ```go
    bodyJSON, err := json.Marshal(req.BodyJSON)
    if err != nil {
        badRequest(c, "ungültiger body_json-Inhalt")
        return
    }
    if err := h.tiptapSvc.ValidateJSON(bodyJSON); err != nil {
        badRequest(c, "nicht erlaubter Editor-Inhalt: "+err.Error())
        return
    }
    bodyHTML, err := h.tiptapSvc.RenderHTML(bodyJSON)
    if err != nil {
        writeInternalErrorResponse(c, "interner serverfehler", err, "HTML-Rendering fehlgeschlagen.")
        return
    }
    bodyText, _ := h.tiptapSvc.ExtractText(bodyJSON)
    ```
