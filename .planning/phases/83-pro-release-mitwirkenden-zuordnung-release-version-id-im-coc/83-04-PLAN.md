```sql
SELECT ac.id AS contribution_id, ac.member_id, m.nickname AS member_display_name,
       ma.file_path AS member_avatar_url,
       COALESCE(ARRAY_AGG(acr.role_code ORDER BY acr.role_code) FILTER (WHERE acr.role_code IS NOT NULL), ARRAY[]::text[]) AS role_codes
FROM anime_contributions ac
JOIN members m ON m.id = ac.member_id
LEFT JOIN media_assets ma ON ma.id = m.avatar_media_id AND ma.status = 'ready'
LEFT JOIN anime_contribution_roles acr ON acr.anime_contribution_id = ac.id
WHERE ac.release_version_id = $1
  AND ac.fansub_group_id = $2
GROUP BY ac.id, m.nickname, ma.file_path
ORDER BY m.nickname ASC
```

Schritt 2 — Fallback (nur wenn Schritt 1 leer): anime-weite Contributions (release_version_id IS NULL):
