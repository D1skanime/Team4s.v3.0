### Package 3: Anime Titles Normalization
**Lane:** Backend (Go)
**Status:** PENDING
**Dependencies:** Package 2

**Scope:**
1. Create migration for `anime_titles` table
2. Create backfill script from existing `anime.title`, `title_de`, `title_en`
3. Update repository to read from normalized table (adapter pattern)

**Migration: anime_titles**
```sql
CREATE TABLE anime_titles (
    id SERIAL PRIMARY KEY,
    anime_id BIGINT NOT NULL REFERENCES anime(id) ON DELETE CASCADE,
    language_id INT NOT NULL REFERENCES languages(id),
    title_type_id INT NOT NULL REFERENCES title_types(id),
    title VARCHAR(500) NOT NULL,
    UNIQUE (anime_id, language_id, title_type_id)
);

CREATE INDEX idx_anime_titles_anime ON anime_titles(anime_id);
CREATE INDEX idx_anime_titles_language ON anime_titles(language_id);
```

**Backfill Logic:**
```
FOR each anime:
  IF title IS NOT NULL:
    INSERT anime_titles (anime_id, language_id=romaji, title_type_id=main, title)
  IF title_de IS NOT NULL:
    INSERT anime_titles (anime_id, language_id=de, title_type_id=official, title_de)
  IF title_en IS NOT NULL:
    INSERT anime_titles (anime_id, language_id=en, title_type_id=official, title_en)
```

### Package 4: Genre Normalization
**Lane:** Backend (Go)
**Status:** PENDING
**Dependencies:** Package 1

**Scope:**
1. Create migration for `genres` table
2. Create migration for `anime_genres` junction table
3. Create backfill script from existing `anime.genre` CSV
4. Update repository to read from normalized table

**Migration: genres**
```sql
CREATE TABLE genres (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100) NOT NULL UNIQUE
);

CREATE INDEX idx_genres_name ON genres(name);
```

**Migration: anime_genres**
```sql
CREATE TABLE anime_genres (
    anime_id BIGINT NOT NULL REFERENCES anime(id) ON DELETE CASCADE,
    genre_id INT NOT NULL REFERENCES genres(id) ON DELETE CASCADE,
    PRIMARY KEY (anime_id, genre_id)
);

CREATE INDEX idx_anime_genres_anime ON anime_genres(anime_id);
CREATE INDEX idx_anime_genres_genre ON anime_genres(genre_id);
```

**Backfill Logic:**
```
1. SELECT DISTINCT genre values from anime table
2. Parse CSV, extract unique genre names
3. INSERT INTO genres for each unique name
4. FOR each anime with genre:
     FOR each genre in CSV:
       INSERT anime_genres (anime_id, genre_id)
```
