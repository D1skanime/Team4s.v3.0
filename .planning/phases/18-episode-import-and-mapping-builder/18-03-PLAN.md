Routes to add:
```text
GET  /api/v1/admin/anime/:id/episode-import/context
POST /api/v1/admin/anime/:id/episode-import/preview
POST /api/v1/admin/anime/:id/episode-import/apply
```

Preview request:
```json
{ "anisearch_id": "1078", "jellyfin_series_id": "optional override", "season_offset": 0 }
```

Apply request:
```json
{
  "canonical_episodes": [{ "episode_number": 9, "title": "..." }],
  "mappings": [{
    "media_item_id": "jellyfin-item",
    "target_episode_numbers": [9, 10],
    "status": "confirmed"
  }]
}
```

Apply must reject `suggested` or `conflict` rows. Skipped rows are allowed and ignored. `season_offset` is suggestion-only; it must not mutate canonical numbering.
</interfaces>
