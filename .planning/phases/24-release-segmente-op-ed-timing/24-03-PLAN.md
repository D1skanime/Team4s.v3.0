**Schritt 4: Backend-Endpunkte per curl testen**

Token aus lokaler Umgebung lesen (AUTH_TOKEN aus .env oder /auth-Endpunkt).

Ein Anime mit vorhandenen Themes ermitteln (z.B. ID aus der DB):
```bash
docker compose exec db psql -U postgres -d team4s -c \
  "SELECT t.anime_id, t.id as theme_id, tt.name FROM themes t JOIN theme_types tt ON tt.id = t.theme_type_id LIMIT 3;"
```

4a. LIST (leer):
```bash
curl -s -H "Authorization: Bearer $TOKEN" \
  "http://localhost:8092/api/v1/admin/anime/{ANIME_ID}/segments" | jq '.data | length'
```
Erwartet: 0 (oder Array)

4b. CREATE:
```bash
curl -s -X POST -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"theme_id": {THEME_ID}, "version": "v1", "start_episode": 1, "end_episode": 12, "start_time": "00:01:30", "end_time": "00:02:30"}' \
  "http://localhost:8092/api/v1/admin/anime/{ANIME_ID}/segments" | jq '.'
```
Erwartet: 201 mit id, theme_type_name, start_episode=1, end_episode=12, start_time="00:01:30"

4c. UPDATE:
```bash
curl -s -X PATCH -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"end_episode": 24}' \
  "http://localhost:8092/api/v1/admin/anime/{ANIME_ID}/segments/{SEGMENT_ID}" | jq '.'
```
Erwartet: 204 No Content

4d. LIST nach Update:
```bash
curl -s -H "Authorization: Bearer $TOKEN" \
  "http://localhost:8092/api/v1/admin/anime/{ANIME_ID}/segments" | jq '.data[0].end_episode'
```
Erwartet: 24

4e. Playback-Query per SQL:
```bash
docker compose exec db psql -U postgres -d team4s -c "
  SELECT t.title, tt.name as type, ts.start_time, ts.end_time
  FROM themes t
  JOIN theme_types tt ON tt.id = t.theme_type_id
  JOIN theme_segments ts ON ts.theme_id = t.id
  WHERE t.anime_id = {ANIME_ID}
    AND ts.version = 'v1'
    AND 5 BETWEEN ts.start_episode AND ts.end_episode
  ORDER BY ts.start_time;
"
```
Erwartet: Zeile mit dem angelegten Segment (Episode 5 liegt zwischen 1 und 24).

4f. DELETE:
```bash
curl -s -X DELETE -H "Authorization: Bearer $TOKEN" \
  "http://localhost:8092/api/v1/admin/anime/{ANIME_ID}/segments/{SEGMENT_ID}"
```
Erwartet: 204
