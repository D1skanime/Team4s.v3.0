             GROUP BY ac.id ORDER BY ac.created_at DESC
             LIMIT 50
  - c.JSON(200, gin.H{"data": items})
  HINWEIS: Diese Abfrage NICHT in einem Repository implementieren — inline als direktes db.Query im Handler ist hier akzeptabel wegen der Zustandsabhaengigkeit von resolveVerifiedMemberID.

|----------|-------------|

| Auth-Boundary → Me-Routes | authMiddleware prueft Token vor Me-Handler-Aufruf |
| Me-Handler → eigene Daten | Ownership-Check verhindert, dass User fremde Eintraege aendern |
