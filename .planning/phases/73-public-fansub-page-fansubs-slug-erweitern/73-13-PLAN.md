**FansubBannerDisplay.tsx** — exakte Struktur:

- Direktive: `'use client'` als erste Zeile.
- Imports: `useEffect`, `useRef`, `useState` aus `react`; `Image` aus `next/image`; `styles` aus `./FansubBannerDisplay.module.css`.
- Typen (lokal, nicht exportiert):
  - `type BannerEdgeFills = { left: string; right: string }`
  - `type BannerSideWidths = { left: number; right: number }`
- Props-Interface: `interface FansubBannerDisplayProps { bannerURL: string; altText?: string }`
- Hilfsfunktionen (Modul-Level, exakt wie Edit-Seite):
  - `function createBannerEdgeFillDataURL(image: HTMLImageElement, side: 'left' | 'right'): string` — sample 1-3px Strip vom linken bzw. rechten Rand, zeichne auf canvas, gib `canvas.toDataURL('image/png')` zurück.
  - `async function loadBannerEdgeFills(imageURL: string): Promise<BannerEdgeFills | null>` — lädt Bild mit `crossOrigin='anonymous'`, ruft beide createBannerEdgeFillDataURL auf, gibt `{ left, right }` zurück; bei Fehler `null`.
- Komponente `FansubBannerDisplay` (named export):
  - State: `bannerEdgeFills: BannerEdgeFills | null` (init null), `bannerSideWidths: BannerSideWidths` (init `{left:0, right:0}`).
  - Refs: `shellRef: useRef<HTMLDivElement | null>(null)`, `imageRef: useRef<HTMLImageElement | null>(null)`.
  - useEffect 1 (dep: `[bannerURL]`): Wenn keine URL → fills auf null setzen + früh return. Sonst `loadBannerEdgeFills(bannerURL)` aufrufen; active-Flag zum Abbrechen; fills setzen.
  - useEffect 2 (dep: `[bannerURL]`): ResizeObserver + `window.addEventListener('resize', measure)` auf shell und image. `measure()` berechnet `left = max(0, round(imageRect.left - shellRect.left) + 8)`, `right = max(0, round(shellRect.right - imageRect.right) + 8)`. Cleanup: disconnect + removeEventListener. Wenn `image.complete` false → `image.addEventListener('load', measure)`.
  - `const showBannerSideFills = bannerSideWidths.left > 12 || bannerSideWidths.right > 12`.
  - Render-Struktur (gleiche DOM-Hierarchie wie Edit-Seite):
    ```
    <div className={styles.bannerShell} ref={shellRef}>
      {/* Left fill */}
      {showBannerSideFills && bannerEdgeFills?.left ? (
        <div
          className={`${styles.bannerSideFill} ${styles.bannerSideFillLeft}`}
          style={{ backgroundImage: `url(${bannerEdgeFills.left})`, width: `${bannerSideWidths.left}px` }}
          aria-hidden="true"
        />
      ) : null}
      {/* Right fill */}
      {showBannerSideFills && bannerEdgeFills?.right ? (
        <div
          className={`${styles.bannerSideFill} ${styles.bannerSideFillRight}`}
          style={{ backgroundImage: `url(${bannerEdgeFills.right})`, width: `${bannerSideWidths.right}px` }}
          aria-hidden="true"
        />
      ) : null}
      {/* Edge fade overlay */}
      {showBannerSideFills ? (
        <div className={styles.bannerEdgeFade} aria-hidden="true" />
      ) : null}
      {/* Centered image */}
      <div className={styles.bannerImage}>
        <Image
          ref={imageRef}
          src={bannerURL}
          alt={altText ?? ''}
          className={styles.bannerImageElement}
          width={1200}
          height={180}
          unoptimized
          priority
        />
      </div>
    </div>
    ```
  - Nur die `backgroundImage` und `width` inline styles sind erlaubt (dynamische Werte); alles andere im CSS-Module.
